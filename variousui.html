Various UI Projects

Here is a collection of various interface design exercises I have conducted.

1. Sign In Screens

A UI Design exercise focusing on creating a simple sign in screen for a conceptual meal sharing platform. My intention was to design an inviting and warm sign in page that matched the wholesome atmosphere of the concept of sharing home cooked meals. The proof of concept designs were created in Adobe XD, and assets were created via Adobe Photoshop.

2. Hawk Solutions

A UI Design and web development exercise to create a website for an electrical solutions company. My intention was to design a concise point of contact page with easy navigation while still providing visitors the ability to view all necessary information, thus the website only consists of a single page. The website was completely from scratch using HTML, CSS and Javascript, while the logo design was conducted in Adobe Illustrator.
